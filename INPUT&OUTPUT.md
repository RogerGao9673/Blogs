# Input && Output ---C++ basics

C++中，选手们有许多输入输出方式：

1.cout&&cin stream:

分别在ostream和istream中，可直接输出。

举例
```cpp
    cout<<"Hello OI";
    cout<<"Hello OI\n" //换行
    cout<<"Hello OI"<<endl //换行
    string a="Hello OI";
    cout<<a;
```
```cpp
    int a;
    cin>>a;
    string b;
    cin>>b;
```
