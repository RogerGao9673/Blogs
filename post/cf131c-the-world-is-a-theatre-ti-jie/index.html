A)分析：

首先，我们必须学会一种叫组合数的东西。

CmnC_{m}^{n}Cmn​ ,表示从mmm个东西里取nnn个，不记顺序（记顺序即为 AmnA_{m}^{n}Amn​ )的情况总数。

这道题目就是许多组合数相加，每一项都是Cboyselected−boys×Cgirlselected−girlsC_{boy}^{selected-boys}\times C_{girl}^{selected-girls}Cboyselected−boys​×Cgirlselected−girls​。

其中，我们枚举selected−boysselected-boysselected−boys的下限到上限，selected−girlsselected-girlsselected−girls就是余下的人。

①selected−boysselected-boysselected−boys的下限： 男生至少444人，但如果女生全选余下的人数还多于444，则男生必须取大于444人，所以男生人数是：max(4,all−girl)max(4,all-girl)max(4,all−girl)；

②selected−boysselected-boysselected−boys的上限：
男生至多all−1all-1all−1人（给女生留111人），但如果男生总人数还不够这么多，那么男生只能取总人数那么多，所以男生人数是：min(all−1,boy)min(all-1,boy)min(all−1,boy)。

#include<cstdio>
#include<iostream>
#include<algorithm>
using namespace std;
long long ans=0;
int boy,girl,all;
inline long long C(int a,int b)//分别表示上文中的n,m
{
    long long ansC=1;//不开long long见祖宗
    for(int i=b-a+1;i<=b;i++) ansC*=i,ansC/=i-b+a;
    return ansC;
} 
int main()
{
    scanf("%d %d %d",&boy,&girl,&all);
    for(int i=max(all-girl,4);i<=min(all-1,boy);i++) 
        ans+=C(i,boy)*C(all-i,girl);
    printf("%I64d\n",ans);//不让用LLD
    return 0;
}
